Processing Latex

So you've gotten some latex by tapping specific tropical trees... congratulations! Latex by itself doesn't do much, but by processing it we can get some very useful byproducts!

If you haven't yet found any trees that produce latex, you can look at the Tapping Index entry for more information!

One of the ways Latex can be used is in the production of Sticky Resin. Sticky resin needs to be produced by treating latex in a Vat or a Pot with some Wood Ash to coagulate it.

Sticky Resin has a number of uses, thanks to it being well... sticky. It can be used to create multiple items such as Tape which will let you move crates around without dropping what's inside, but more importantly, it can be used to create Resin Circuit Boards and Resistors, which will be some of your first electrical components.

The other main use of Latex is to make rubber. To make the latex stronger and harder for industrial uses, it needs to be heated (either in a Vat or a Pot) with added Sulfur Powder to produce Vulcanized Latex. In fact, this whole process is called vulcanization!

Using a Extractor or Vacuum Chamber, the Vulcanized Latex can be dried into Raw Rubber Pulp. This pulp needs to be further treated with Sulfur Dust in an Alloy Smelter to create Rubber Ingots.

Rubber Ingots have a number of very useful applications, such as for crafting Belts for your factories or for insulating electrical wires. Insulated wires become safe to touch without accidentally electrocuting yourself, and have reduced voltage loss.

Sticky Resin can also be obtained via Conifer Pitch tapped from certain Coniferous Trees, but these trees cannot be used for Rubber.
